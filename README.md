# urls_downloader

### Description

Program downloads files specified in `files_urls.txt`.
 
### Build

  `javac *.java`
 
### Usage

  `java Program --threadsCount=N` where `N` is count of workong threads.

For example `java Program --threadsCount=3` for 5 file urls to precess:
